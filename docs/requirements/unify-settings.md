# Unify Settings

使用 $huashu 统一设置界面

* 将现有 设置主题的按钮改成 系统设置 按钮

* 设置界面左右两栏,左边菜单,右边对应设置panel

* 菜单

  * AI

    * 设置当前AI CLI, 先固定下拉框 (Codex, Claude Code), 选择或者输入对应默认model name

    * 这些选项要check过 用户电脑有这些CLI 才能显示

* Theme: 将当前Theme Popup 内容集成到这边

  * 外面Theme Switch button点击, 默认切换到 Theme 的Menu

* check update: 显示当前版本信息, 有个检测更新按钮
