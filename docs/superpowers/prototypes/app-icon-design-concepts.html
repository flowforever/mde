MDE App Icon · Design Exploration

一个像“本地文稿工作台”的 Markdown 图标

假设：MDE 的识别资产来自真实产品，而不是外部品牌规范。核心线索是暖纸编辑器、左侧资源管理器、本地文件、Markdown 语义和克制的工具感。 这页先给 3 个可落地母版，每个都检查 128/96/64/44/32 像素可读性。

叙事角色 启动器身份。它应该让人想起“打开一个本地 Markdown 工作区”，而不是展示完整功能列表。
观众距离 Dock、Alt-Tab、安装包和发布页都要用。32 像素仍能读出轮廓，1024 像素有细节。
视觉温度 安静、可信、偏写作工具。沿用 Manuscript 主题的纸色，但给深色 Dock 留一个对比方案。
容量估算 一个图标只放一个主隐喻：文稿。资源管理器、Markdown、命令能力只做二级符号。
