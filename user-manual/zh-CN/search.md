# 搜索

MDE 提供当前文档搜索和工作区全局搜索。

## 当前文档搜索

点击编辑器标题栏的“搜索当前 Markdown”按钮，或使用快捷键打开当前文档搜索。输入关键字后，匹配项会在编辑器中高亮，回车会保存搜索记录并跳转到下一处。

再次聚焦搜索框时，MDE 会显示本机保存的搜索记录；输入过程中只保留匹配当前输入的记录。记录右侧的固定按钮可以把该关键字的所有命中以独立颜色高亮，方便同时对比多个关键字。已固定关键字会显示在搜索记录上方，可以随时删除。

![当前文档搜索](/screenshots/zh-CN/editor-search.png)

## 工作区全局搜索

点击 Explorer 顶部的“搜索工作区内容”按钮，输入关键字后，MDE 会搜索当前工作区中的 Markdown 文件。点击结果会打开对应文件并定位到命中内容。

打开搜索框时会列出本机保存的工作区搜索记录。点击记录会重新执行对应搜索，方便回到常用查询。

![工作区搜索](/screenshots/zh-CN/workspace-search.png)

搜索范围只包括当前工作区下的 Markdown 内容，MDE 管理目录 `.mde/` 不会作为普通内容搜索。
