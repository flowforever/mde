# 搜索

MDE 提供当前文档搜索和工作区全局搜索。

## 当前文档搜索

点击编辑器标题栏的“搜索当前 Markdown”按钮，或使用快捷键打开当前文档搜索。输入关键字后，匹配项会在编辑器中高亮，回车会保存搜索记录并跳转到下一处。

再次聚焦搜索框时，MDE 会显示本机保存的搜索记录；输入过程中只保留匹配当前输入的记录。记录右侧的固定按钮可以把该关键字的所有命中以独立颜色高亮，方便同时对比多个关键字。已固定关键字会显示在搜索记录上方，可以随时删除。

![当前文档搜索](/screenshots/zh-CN/editor-search.png)

## 工作区全局搜索

点击 Explorer 顶部的“搜索工作区内容”按钮，输入关键字后，MDE 会搜索当前工作区中的 Markdown 文件。结果预览会高亮匹配到的关键字，点击结果会打开对应文件并定位到命中内容。

搜索弹窗右上角可以切换搜索模式，鼠标悬停在模式按钮上会显示对应快捷键。放大镜模式搜索 Markdown 正文；`@` 模式搜索当前工作区中的 Markdown 文件路径。也可以使用 `Cmd+P` / `Ctrl+P` 直接打开路径搜索；搜索弹窗打开后重复按这个快捷键，会在正文搜索和路径搜索之间轮流切换。路径搜索支持类似 VS Code 的 `/` 分段模糊匹配，例如输入 `doc/nst/dp` 可以匹配 `docs/nested/deep.md`。路径搜索结果会显示匹配的文件路径，点击结果会直接打开对应文件，不会把路径关键字带入当前文档搜索。

打开搜索框时会以标签形式列出本机保存的工作区搜索记录，最多显示 16 条。输入过程中，标签会筛选为匹配当前输入的记录；没有匹配记录时，标签区域会隐藏。点击标签会重新执行对应搜索，并让输入框保持聚焦，方便继续编辑查询。

![工作区搜索](/screenshots/zh-CN/workspace-search.png)

搜索范围只包括当前工作区下的 Markdown 内容，MDE 管理目录 `.mde/` 不会作为普通内容搜索。
