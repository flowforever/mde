# 设置

点击 Explorer 底部的设置按钮可以打开设置页。

![主题设置](/screenshots/zh-CN/settings-theme.png)

## Preference

Preference 分区可以切换应用语言，也可以使用本机 AI CLI 生成自定义应用语言包。自定义语言包会在语言下拉框中标记为自定义；选择自定义语言包后，可以用 AI 根据最新应用文案更新当前语言包。

## Theme

Theme 分区可以选择跟随系统外观，也可以分别配置浅色和深色主题。主题选择会保存在本机。

## AI

AI 分区用于选择本地 AI CLI 和默认模型名。这里只会显示当前机器上可以检测到的 CLI。

## Check Update

Check Update 分区显示当前 MDE 版本，并检查 GitHub Releases 上的新版本。更新检查失败通常和网络、GitHub API 限制或当前运行环境有关。
