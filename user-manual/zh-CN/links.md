# 链接

MDE 支持在编辑器中插入 Markdown 文件链接和外部网页链接。

## 插入已有文档链接

在编辑器中输入 `/`，选择“链接”，然后搜索目标 Markdown 文件。选择建议后，MDE 会插入相对路径链接。

![插入链接](/screenshots/zh-CN/insert-link.png)

## 创建新文档链接

链接弹窗中可以切换到“新建文档”，选择目录并输入文件名。MDE 会创建目标文档并插入链接。

## 打开链接

* 同一工作区内的 Markdown 链接会在当前窗口打开。
* 已知其它工作区中的 Markdown 链接会在新窗口打开。
* HTTP/HTTPS 链接会交给系统默认浏览器。
* 出于安全原因，MDE 不支持 `javascript:` 链接。
