# 工作区与文件

左侧 Explorer 展示当前工作区的 Markdown 文件和包含 Markdown 文档的文件夹。目录会排在 Markdown 文件前面；点号开头的隐藏条目、没有 Markdown 文档的文件夹默认不显示。打开大型工作区时，MDE 会先加载根目录，展开文件夹时再按需读取子目录。

![工作区资源管理器](/screenshots/zh-CN/workspace-explorer.png)

## 常用操作

Explorer 顶部工具栏提供：

| 操作 | 用途 |
| --- | --- |
| 新建 Markdown 文件 | 在当前工作区或选中目录下创建 `.md` 文件 |
| 新建文件夹 | 在当前工作区或选中目录下创建文件夹 |
| 恢复已删除文档 | 展开“已删除文档”列表，选择可恢复的历史版本 |
| 显示隐藏条目 | 临时显示点号开头、没有 Markdown 文档或手动隐藏的条目 |
| 刷新 | 重新读取工作区文件树 |

右键文件或文件夹可以复制、粘贴、复制相对路径、复制绝对路径、重命名、删除，也可以在文件夹内创建新文件或新文件夹。选中树中的文件或文件夹后，也可以使用 `Command/Ctrl+C` 复制，再选中目标文件夹使用 `Command/Ctrl+V` 粘贴。

复制 Markdown 文件时，MDE 会一起复制文档引用的本地图片资源。把 Finder 或系统文件管理器里的 Markdown 文件粘贴到 Explorer 时，MDE 会把相对图片资源迁移到目标文档旁边的 `.mde/assets` 目录。删除前 MDE 会显示确认，生产环境中的 MDE 删除会移动到系统回收站。

## 拖拽打开

你可以把 Finder、Explorer 或系统文件管理器中的文件夹和 Markdown 文件拖到 MDE 主窗口中。拖拽悬停时，窗口会出现轻量的放置提示，松手后 MDE 会打开第一个有效的本机资源。

拖入当前工作区内的 Markdown 文件时，MDE 会在当前窗口打开它，并展开、定位左侧 Explorer 中的对应条目。拖入当前工作区内的文件夹时，MDE 会刷新文件树并选中该文件夹。拖入当前工作区根目录会刷新当前文件树，不会新开窗口。

拖入当前工作区之外的 Markdown 文件或文件夹时，如果当前窗口已经打开工作区或单个 Markdown 文件，MDE 会保留当前窗口内容，并在新窗口打开拖入资源。空窗口会直接在当前窗口打开拖入资源。非 Markdown 文件不会作为文档打开。

## Recent Files

Recent Files 记录当前工作区最近打开的 Markdown 文件，便于快速回到刚看过的文档。它不会保存文档内容，也不会替代版本历史。

展开 Recent Files 时，MDE 会校验列表里的 Markdown 文件是否仍在磁盘上；已经被 Finder 或其它工具删除的文件会从列表中移除。

## 隐藏条目

你可以通过右键菜单隐藏某个条目。隐藏状态按工作区保存；点击“显示隐藏条目”可以临时查看并恢复显示。没有 Markdown 文档的文件夹只会在显示隐藏条目时出现。
