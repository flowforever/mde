# 工作区与文件

左侧 Explorer 展示当前工作区的 Markdown 文件和文件夹。目录会排在 Markdown 文件前面，点号开头的隐藏条目默认不显示。

![工作区资源管理器](/screenshots/zh-CN/workspace-explorer.png)

## 常用操作

Explorer 顶部工具栏提供：

| 操作 | 用途 |
| --- | --- |
| 新建 Markdown 文件 | 在当前工作区或选中目录下创建 `.md` 文件 |
| 新建文件夹 | 在当前工作区或选中目录下创建文件夹 |
| 恢复已删除文档 | 展开“已删除文档”列表，选择可恢复的历史版本 |
| 显示隐藏条目 | 临时显示点号开头或手动隐藏的条目 |
| 刷新 | 重新读取工作区文件树 |

右键文件或文件夹可以复制、粘贴、复制相对路径、复制绝对路径、重命名、删除，也可以在文件夹内创建新文件或新文件夹。选中树中的文件或文件夹后，也可以使用 `Command/Ctrl+C` 复制，再选中目标文件夹使用 `Command/Ctrl+V` 粘贴。

复制 Markdown 文件时，MDE 会一起复制文档引用的本地图片资源。把 Finder 或系统文件管理器里的 Markdown 文件粘贴到 Explorer 时，MDE 会把相对图片资源迁移到目标文档旁边的 `.mde/assets` 目录。删除前 MDE 会显示确认，生产环境中的 MDE 删除会移动到系统回收站。

## Recent Files

Recent Files 记录当前工作区最近打开的 Markdown 文件，便于快速回到刚看过的文档。它不会保存文档内容，也不会替代版本历史。

## 隐藏条目

你可以通过右键菜单隐藏某个条目。隐藏状态按工作区保存；点击“显示隐藏条目”可以临时查看并恢复显示。
