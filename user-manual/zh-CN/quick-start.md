# 快速开始

第一次启动 MDE 时，会看到工作区管理弹窗。你可以打开一个文件夹工作区，也可以只打开单个 Markdown 文件。

![打开工作区入口](/screenshots/zh-CN/quick-start-open-workspace.png)

## 打开文件夹工作区

1. 点击“打开新工作区”。
2. 选择一个本地文件夹。
3. 在左侧 Explorer 中选择 Markdown 文件开始编辑。

文件夹工作区适合管理一组文档、使用全局搜索、内部链接和 AI 生成结果缓存。

## 打开单个 Markdown 文件

如果只想快速编辑一个文件，点击“打开 Markdown 文件”。MDE 会把这个文件所在目录作为临时工作区，并在工作区管理里记住它。

## 最近资源

工作区管理会显示最近打开过的工作区和文件。左侧 Explorer 的“最近文件”只用于当前工作区内的快速跳转，不是文档版本历史。

## 拖拽打开

你可以把文件夹或 Markdown 文件拖到 MDE 窗口中。当前已经打开工作区时，拖入的外部资源会在新窗口打开。

## 从命令行打开

安装版 MDE 启动后会在后台检测 `mde` 命令是否可用。注册成功后，可以在终端中使用 `mde /path/to/workspace` 打开文件夹，也可以使用 `mde /path/to/file.md` 打开单个 Markdown 文件。已经存在可执行 `mde` 命令时，MDE 不会覆盖它。
