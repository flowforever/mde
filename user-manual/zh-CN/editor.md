# 编辑 Markdown

选择 Markdown 文件后，MDE 会在主编辑区打开它。编辑器会自动保存，保存中和未保存状态会显示在标题栏。

![编辑器主界面](/screenshots/zh-CN/editor-main.png)

## 自动保存

MDE 会在空闲后自动保存当前文档。手动切换文件、离开编辑器或触发部分编辑器操作时，也会尝试保存最新内容。

如果自动保存检测到会把非空文档清空，会要求确认。取消后编辑器和磁盘文件都会保留原内容。

正文中的连续空行会在保存和重新打开后保留；代码块内部的空行不会被改写。

## 视图宽度

编辑器标题栏右侧可以在居中编辑视图和全宽编辑视图之间切换。这个偏好会保存在本机。

当右上角可用操作超过 5 个时，MDE 会优先保留搜索、AI 翻译、AI 摘要和版本历史等高频入口，并用箭头展开行距和视图宽度等其余操作。

## YAML Frontmatter

文件开头的 YAML frontmatter 会显示在正文上方的轻量信息行里，不会进入正文编辑区。点击信息行可以查看字段视图；点击“源码”可以查看或编辑原始 YAML。YAML 解析失败时，MDE 会直接显示源码和保留提示，保存正文不会改写这段 YAML。

## 粘贴图片

粘贴图片时，MDE 会把图片保存到当前工作区的 `.mde/assets/`，并在 Markdown 中插入相对引用。

如果 Markdown 文件移动后原来的 `.mde/assets/` 相对图片失效，MDE 会在打开文件时从当前工作区其他 `.mde/assets/` 位置查找唯一匹配的图片，并自动复制到当前文档旁边。修复完成后，编辑器会显示一条轻量状态提示。

## 代码块

代码块左上角会显示语言选择器。选择语言后，MDE 会按对应语言显示语法高亮；常见简写语言名会自动归一到可选语言。语法高亮会跟随当前主题保持可读。

## Mermaid 流程图

使用 `mermaid` 代码块可以渲染流程图。源码下方会显示完整缩略预览，保留原始代码块供继续编辑。点击缩略图可以打开更大的只读预览；预览窗口支持拖动画布、双指滑动平移、双指张合或按钮缩放、重置视图、切换整页显示，并且可以选中流程图里的文字。渲染失败时，MDE 会显示错误状态并保留原始源码。

![Mermaid 流程图](/screenshots/zh-CN/mermaid-flowchart.png)

## 文档版本历史

编辑器标题栏的“版本历史”按钮可以打开或关闭右侧恢复面板。你可以预览只读历史版本，并把当前文档恢复到选中的版本。恢复前，MDE 会先把当前内容保存进历史。

Explorer 顶部的恢复入口用于找回已经删除的文档。它会展开“已删除文档”列表，不会改变“最近文件”的展开状态。

## Markdown round-trip 限制

MDE 以 Markdown 为真实存储格式。部分未来富文本块如果无法无损映射到 Markdown，保存后可能以最接近的 Markdown 形式保留。
